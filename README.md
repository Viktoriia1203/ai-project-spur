# Text Similarity Project

This project demonstrates a simple text similarity system using embeddings and cosine similarity.

## Description

The program compares a query sentence with a list of documents and finds the most similar one.

Due to API limitations, embeddings are simulated locally, but the similarity pipeline remains the same as in real-world applications.

## How it works

1. Convert text into vector representations (embeddings)
2. Compute cosine similarity between vectors
3. Return the most similar document
